# dex_analyze

analyze dex structure 

## usage

```shell
usage:python3 dex_analyze.py [-h] [-i DEX_PATH] [--all] [--header] [--string] [--type] [--proto] [--field] [--method] [--class] [--map]

purpose: analyze dex

optional arguments:
  -h, --help   show this help message and exit
  -i DEX_PATH  input file path
  --all        show all
  --header     show header
  --string     show string
  --type       show type
  --proto      show proto
  --field      show field
  --method     show method
  --class      show class
  --map        show map

information end
```
